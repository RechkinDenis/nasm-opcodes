ADD

Описание: Складывает операнд-источник с операндом-назначением и сохраняет результат в операнде-назначении.

Синтаксис: ADD destination, source

Пример: ADD AX, BX

CALL

Описание: Вызывает процедуру, помещая адрес возврата в стек и переходя к процедуре.

Синтаксис: CALL procedure

Пример: CALL myProcedure

CMP

Описание: Сравнивает два операнда, вычитая источник из назначения, устанавливая флаги, но не сохраняя результат.

Синтаксис: CMP destination, source

Пример: CMP AX, BX

DEC

Описание: Уменьшает операнд-назначение на 1.

Синтаксис: DEC destination

Пример: DEC AX

DIV

Описание: Делит операнд-назначение на операнд-источник, сохраняя результат в аккумуляторе, а остаток — в регистре остатка.

Синтаксис: DIV source

Пример: DIV BX

HLT

Описание: Останавливает процессор до тех пор, пока не произойдет сброс или прерывание.

Синтаксис: HLT

Пример: HLT

IDIV

Описание: Делит знаковые целые числа, сохраняя частное и остаток в определенных регистрах.

Синтаксис: IDIV source

Пример: IDIV BX

IMUL

Описание: Умножает знаковые целые числа.

Синтаксис: IMUL destination, source

Пример: IMUL AX, BX

INC

Описание: Увеличивает операнд-назначение на 1.

Синтаксис: INC destination

Пример: INC AX

INT

Описание: Вызывает программное прерывание.

Синтаксис: INT interrupt_number

Пример: INT 21h

INTO

Описание: Прерывает выполнение программы, если установлен флаг переполнения.

Синтаксис: INTO

Пример: INTO

IRET

Описание: Возвращается из обработчика прерываний, извлекая IP, CS и флаги из стека.

Синтаксис: IRET

Пример: IRET

JNZ

Описание: Переходит по указанному адресу, если флаг нуля не установлен.

Синтаксис: JNZ label

Пример: JNZ myLabel

JMP

Описание: Безусловно переходит по указанному адресу.

Синтаксис: JMP label

Пример: JMP myLabel

LEA

Описание: Загружает эффективный адрес операнда-источника в регистр назначения.

Синтаксис: LEA destination, source

Пример: LEA AX, [BX+SI]

MOV

Описание: Перемещает данные из операнда-источника в операнд-назначение.

Синтаксис: MOV destination, source

Пример: MOV AX, BX

MUL

Описание: Умножает беззнаковые целые числа.

Синтаксис: MUL source

Пример: MUL BX

POP

Описание: Извлекает верхнее значение из стека в операнд-назначение.

Синтаксис: POP destination

Пример: POP AX

PUSH

Описание: Помещает операнд-источник в стек.

Синтаксис: PUSH source

Пример: PUSH AX

RET

Описание: Возвращается из процедуры, извлекая адрес возврата из стека.

Синтаксис: RET

Пример: RET

RETN

Описание: То же, что и RET, но используется в некоторых специфических контекстах для возврата из процедур.

Синтаксис: RETN

Пример: RETN

SUB

Описание: Вычитает операнд-источник из операнда-назначения и сохраняет результат в назначении.

Синтаксис: SUB destination, source

Пример: SUB AX, BX

XOR

Описание: Выполняет побитовую операцию XOR между операндом-источником и операндом-назначением, сохраняя результат в назначении.

Синтаксис: XOR destination, source

Пример: XOR AX, BX
